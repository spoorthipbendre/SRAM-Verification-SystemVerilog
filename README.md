# SRAM Verification using SystemVerilog

This project focuses on the functional verification of SRAM using SystemVerilog methodology. The verification environment validates memory read/write operations through simulation, debugging, and waveform analysis using QuestaSim/ModelSim.

## Features
• SRAM Read/Write Verification
• SystemVerilog Testbench
• Simulation and Debugging
• Waveform Analysis

## Tools Used
• SystemVerilog
• QuestaSim / ModelSim

## Verification
Verified SRAM functionality through simulation and memory operation testing.
